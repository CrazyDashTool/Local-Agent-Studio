# Telegram Connector

Draft plugin for Local Agent Studio.

## What it adds

- Search Telegram messages
- Summarize a chat
- Draft replies without sending by default

## Install test

Push this folder to a GitHub repository and install it from Local Agent Studio with:

```text
https://github.com/your-name/las-telegram-connector
```

The app downloads `plugin.json` first and shows requested permissions before enabling the plugin.
